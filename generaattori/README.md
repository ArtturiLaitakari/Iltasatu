# Iltasatu Hahmonluonti Generaattori

Tämä on hahmonluonti generaattori Iltasatu roolipeliin joka on FATE pohjainen peli mutta optimoitu helpoksi pelinjohtajalle ja joustavaksi pelaajille.

## Ominaisuudet

- Automaattinen hahmon luonti
- FATE-pohjaiset säännöt
- Helppokäyttöinen käyttöliittymä
- Joustava pelaajille

## Käyttöönotto

```bash
npm install
npm run dev
```

Sovellus käynnistyy osoitteessa http://localhost:1974/

## Kehitys

- React + Vite
- Suomenkielinen koodi ja käyttöliittymä
